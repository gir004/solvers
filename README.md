# Solvers (made in SochiSirius2020)

## Chess
To run *chess.py* use:

``` bash
python3 chess.py
8
```

or

``` bash
python3 chess.py 8
```

Output format:

```
.......*
...*....
*.......
..*.....
.....*..
.*......
......*.
....*...
```

## Sudoku
To run *sudoku.py* use:

``` python3
python3 sudoku.py
```
And enter sudoku in format:
```
5****34*7
***62****
****54**1
*7****8*6
**3***5**
8*5****2*
1**37****
****98***
4*81****5
```
